# Style Library

Use this library to avoid repeating one editing template. Pick one style direction per video, then adapt it to the transcript and brand.

## 1. Creator Perk Explainer

Use for Instagram-style offer, tutorial, tool, and deal reels.

- Pacing: fast hook in 0-3 seconds, proof screen by 5-10 seconds, steps every 2-4 seconds.
- Layout: full talking head or screen-recording punches, with floating proof cards.
- Captions: large high-contrast phrase captions; important words can pop bigger.
- Graphics: checklist cards, benefit pills, pricing slash, arrows, step numbers, scrolling ticker.
- Motion: quick slide-ins, scale pops, cursor/click highlights, subtle zoom punches.
- Palette: 2-3 colors, usually dark/white plus one bright accent.

## 2. HyperFrames / Agentic Editing Demo

Use for AI editing workflow videos and technical tool demos.

- Pacing: show pipeline first, then tool setup, then preview/iterate/render.
- Layout: split between code/editor UI, timeline, preview window, and final video result.
- Captions: clean labels, step headers, command snippets, small callouts.
- Graphics: node pipeline, file tree, terminal blocks, storyboard cards, render status.
- Motion: mask reveals, horizontal wipes, progress bars, browser-window movement.
- Palette: restrained dark UI with one accent color.

## 3. Premium Business Split-Screen

Use for AI, SaaS, business education, founder commentary, and comparison videos.

- Pacing: one scene per 3-4 seconds.
- Layout: top motion graphics, bottom talking head, captions in the middle.
- Captions: phrase-level, bold, centered, strong shadow or stroke.
- Graphics: comparison cards, repo scans, dashboards, risk matrix, metrics, workflow loop.
- Motion: smooth slides, scale reveals, graph builds, cursor-like emphasis.
- Palette: black/white plus one accent, or white/black plus one accent.

## 4. Screen-Proof Tutorial

Use when the audience must trust that a process is real.

- Pacing: screen proof appears early and stays frequent.
- Layout: screen recording as the main visual with speaker as small picture-in-picture or lower half.
- Captions: short commands and step labels.
- Graphics: numbered steps, cursor rings, click flashes, zoomed UI crops.
- Motion: zoom into important interface areas, highlight boxes, checklist completion.
- Palette: match the source UI with one brand accent.

## 5. High-Retention SaaS Ad

Use for product promos, lead-gen tools, AI tools, and landing-page style videos.

- Pacing: visual change every 1.5-3 seconds.
- Layout: UI mockups, dashboards, product cards, motion typography.
- Captions: minimal and polished, more like launch-video typography than subtitles.
- Graphics: product screens, metric counters, benefit cards, before-after panels.
- Motion: polished mask reveals, parallax cards, soft shadows, light sweeps.
- Palette: product brand colors only; avoid rainbow accents.

## Variation Rule

Before rendering, ask: "Does this look like the last video?" If yes, change at least three of:

- layout
- palette
- caption type
- graphic vocabulary
- transition language
- pacing
- sound design
